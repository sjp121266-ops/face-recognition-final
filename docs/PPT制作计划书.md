### 第 9 页：核心识别流程

| 区块 | 内容 |
| --- | --- |
| 标题 | **核心识别流程** |

放一个流程图：

```
CameraX 拍照
    ↓
ML Kit 人脸检测
    ↓ (单张人脸 + 质量检查)
人脸裁剪（统一扩边策略）
    ↓
FaceNet TFLite 推理（含 TTA 增强）
    ↓
128 维特征向量
    ↓
与本地人脸库逐人计算 L2 距离
    ↓
综合 L2 分数 = minDist + (avgDist-minDist) × 0.35
    ↓
判定：score < 10.0 → 识别成功 / score ≥ 10.0 → 未知人员
```

### 第 10 页：FaceNet 原理简述

| 区块 | 内容 |
| --- | --- |
| 标题 | **FaceNet：把人脸映射到 128 维欧氏空间** |
| 核心思想 | 用 Triplet Loss 训练，让同一人距离近、不同人距离远 |
| 公式 | `L = max(0, d(A,P) - d(A,N) + α)` |
| 图解 | 放一张示意：Anchor 和 Positive 靠近，和 Negative 推远 |
| 输入输出 | 160×160×3 RGB → CNN → 128-dim L2 normed vector |

### 第 11 页：关键技术 — TTA + 直方图均衡化

| 区块 | 内容 |
| --- | --- |
| 标题 | **推理策略优化：TTA + 直方图均衡化** |
| TTA | 原图推理 + 水平翻转推理 → 取平均值 |
| 效果 | 约 1-3% 准确率提升，角度变化更鲁棒 |
| 直方图均衡化 | TFLite TensorOperator 在推理前做光照归一化 |
| 效果 | 过暗/过曝场景下特征一致性提升 |
| 关键 | 不换模型、不重新训练，仅优化推理策略 |

### 第 15 页：关键技术 — 人脸库健康分析

| 区块 | 内容 |
| --- | --- |
| 标题 | **人脸库健康分析 + 自适应阈值校准** |
| 检查项 1 | 异常向量清理（非 128 维 / NaN / Infinity） |
| 检查项 2 | 同人一致性（同人最大 L2 距离） |
| 检查项 3 | 跨人区分度（跨人最近 L2 距离） |
| 校准 | 同人最大 L2 = 7.2，跨人最近 L2 = 14.8 → 阈值建议 10~12 |
| 价值 | 帮助演示者科学解释"为什么用 10.0" |
